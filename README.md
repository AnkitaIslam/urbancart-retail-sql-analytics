# UrbanCart Retail Shop Analytics — SQL Project ## Project Overview A retail analytics project using PostgreSQL to analyze 1,200 orders, 100 customers, 41 products across 10 cities in Bangladesh. ## Business Questions Answered (Q1–Q25) - Revenue analysis by city, category, and product - Customer behavior and purchasing patterns - Payment method preferences - Stock risk analysis - Order completion and cancellation rates ## Tools Used - PostgreSQL (pgAdmin 4) - Microsoft Excel (charts) - Canva (presentation) ## Database Schema 5 tables: FactOrders, FactOrderItems, DimCustomers, DimProducts, FactPayment ## Key Findings - Total Revenue: BDT 22,45,122 - Top City: Barishal (173 orders) - Top Product: Power Bank 10000mAh (BDT 304,000) - Completion Rate: 59.4% - Top Payment: Cash on Delivery (488 orders) ## Files | File | Description | |------|-------------| | urbancart_queries.sql | All 25 SQL queries | | UrbanCart_Presentation.pdf | Slides with results | | SQL_Queries_Documentation.pdf | Detailed query doc | | er_diagram.png | Database schema diagram |
